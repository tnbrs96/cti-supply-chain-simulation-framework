# CTI Supply Chain Simulation Framework

## Overview
This project presents a Cyber Threat Intelligence (CTI) simulation framework designed to model supply chain risk scenarios through structured and reproducible synthetic data generation.

The framework is intended for:
- CTI training and analytical exercises
- Threat modeling and scenario simulation
- Validation of reporting workflows
- Supply chain risk assessment prototyping

## Features
- Simulated CVE generation with CVSS scoring
- IOC generation (IP, domain, hash, URL, email)
- MITRE ATT&CK mapping
- Risk scoring and classification
- Automated report generation (Word)
- Structured output (Excel dataset)
- Visual analytics (charts)

## Methodology
The framework operates in a simulation-based mode:
- Data is synthetically generated using deterministic logic
- No real-time threat feeds are used
- Outputs are reproducible and consistent

## Use Cases
- Cyber Threat Intelligence training
- Tabletop exercises
- Supply chain risk modeling
- Analytical workflow validation

## Limitations
- Data is simulated and does not reflect real-world incidents
- Not intended for operational threat detection
- No integration with live CTI feeds (e.g. MISP, VirusTotal)
